## üöÄ Getting Started

### Prerequisites

- Windows 10/11 (x64)
- Visual Studio 2022
- .NET 8.0
- Internet connection for auto-updating offsets

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/xfi0/Titled-Gui-CS2.git
   cd Titled-Gui-CS2
   ```
  2. Build / run the solution in debug mode (x64) 
  3. Run TitledGui.exe as Administrator
